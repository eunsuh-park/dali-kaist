===========================================
ACTIVITY ENTITY (활동 엔티티)
===========================================

활동/뉴스 데이터 및 관련 컴포넌트를 관리하는 엔티티입니다.

파일 목록:
----------
activity/
├── model.js              # 활동 데이터 모델 (향후 추가)
└── activity-card.js      # 활동 카드 컴포넌트 (향후 추가)

각 파일의 역할:
--------------
1. model.js (향후 추가)
   - 활동 데이터 구조 정의
   - 데이터 로드 함수
   - 데이터 검증 함수

2. activity-card.js (향후 추가)
   - 활동 카드 UI 생성 함수
   - 재사용 가능한 컴포넌트

현재 상태:
----------
- 활동 데이터는 현재 pages/activity/activity.html에 하드코딩되어 있음
- 향후 데이터를 entities/activity/data.js로 이동 예정
- 활동 카드 컴포넌트를 분리하여 재사용성 향상 예정

주의사항:
---------
- 활동 엔티티는 아직 완전히 구현되지 않음
- 향후 데이터 구조를 정의하고 모델을 추가해야 함
